# 🐍 Snake Game

基于 HTML5 Canvas 的纯前端贪吃蛇游戏，零外部依赖，浏览器直接打开即玩。支持桌面键盘操作与手机触屏操作。

## 功能特性

- 🎮 **4 种难度**：简单 / 中级 / 困难 / 地狱，不同墙体数量与速度
- 🧱 **墙体障碍**：随机生成连通墙体线段，每局不同
- 🏆 **排行榜**：localStorage 持久化 Top 10，含难度标签与时间戳
- 🔊 **音效反馈**：Web Audio API 合成音效（吃食物 / 死亡 / 开始），无需音频文件
- ⏸️ **暂停继续**：P / Esc 键暂停，再次按下继续；触屏右上角按钮暂停
- 📱 **触屏支持**：滑动控制方向，开始 / 退出按钮，手机浏览器完整可玩
- 📱 **窗口自适应**：25×25 逻辑网格，canvas 大小随窗口自动调整
- 🏅 **通关条件**：蛇身填满整个网格即为通关

## 快速开始

```bash
# 方式一：直接打开
open snake-game.html

# 方式二：启动本地服务器（手机测试）
npm run serve
# 手机浏览器访问 http://<局域网IP>:8080/snake-game.html
```

无需安装任何外部依赖；`package.json` 只提供脚本入口。也可以不用 npm，直接运行：

```bash
python3 -m http.server 8080 --bind 0.0.0.0
```

## 操作说明

### 桌面端（键盘）

| 按键 | 操作 |
|------|------|
| ↑ ↓ ← → / W A S D | 移动方向 / 开始游戏 |
| 1 2 3 4 | 切换难度（初始界面） |
| P / Esc | 暂停 / 继续 |
| 空格 | 重新开始 |

### 手机端（触屏）

| 手势 / 按钮 | 操作 |
|-------------|------|
| 在游戏区域**滑动** | 控制方向 |
| 点按「开始游戏」按钮 | 开始游戏 |
| 点按 ⏸️ 按钮（右上角） | 暂停 / 继续 |
| 点按 ✕ 按钮（左上角） | 退出游戏，回到初始界面 |
| 点按难度按钮 | 切换难度（初始界面） |
| 点按「重新开始」按钮 | 游戏结束后重新开始 |

## 难度配置

| 难度 | 墙体段数 | 速度 |
|------|----------|------|
| 简单 | 0 | 150ms |
| 中级 | 1 | 130ms |
| 困难 | 2 | 110ms |
| 地狱 | 3 | 85ms |

| 触屏 | Touch Events（touchstart/touchend） |
| 存储 | localStorage |
| 依赖 | **零外部依赖** |
