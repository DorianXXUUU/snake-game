# 🐍 Snake Game

基于 HTML5 Canvas 的纯前端贪吃蛇游戏，零外部依赖，浏览器直接打开即玩。

## 功能特性

- 🎮 **4 种难度**：简单 / 中级 / 困难 / 地狱，不同墙体数量与速度
- 🧱 **墙体障碍**：随机生成连通墙体线段，每局不同
- 🏆 **排行榜**：localStorage 持久化 Top 10，含难度标签与时间戳
- 🔊 **音效反馈**：Web Audio API 合成音效（吃食物 / 死亡 / 开始），无需音频文件
- ⏸️ **暂停继续**：P / Esc 键暂停，再次按下继续
- 📱 **窗口自适应**：25×25 逻辑网格，canvas 大小随窗口自动调整
- 🏅 **通关条件**：蛇身填满整个网格即为通关

## 快速开始

```bash
# 方式一：直接打开
open snake-game.html

# 方式二：启动本地服务器（可选）
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080/snake-game.html
```

无需安装任何依赖。

## 操作说明

| 按键 | 操作 |
|------|------|
| ↑ ↓ ← → / W A S D | 移动方向 |
| 1 2 3 4 | 切换难度（初始界面） |
| P / Esc | 暂停 / 继续 |
| 空格 | 重新开始 |

## 难度配置

| 难度 | 墙体段数 | 速度 |
|------|----------|------|
| 简单 | 0 | 150ms |
| 中级 | 1 | 130ms |
| 困难 | 2 | 110ms |
| 地狱 | 3 | 85ms |
